Ko2D Engine
------------------
Ko2D is a cross-platform game engine writen in pure C++ using OpenGL ES 2. It's name is 2D engine but it can do anything including 3D games.
There are 2 template for Windows and Android version located in Build folder.
For Android version, build using NDK and ANT. Then copy Resource folder to /mnt/sdcard/KO2D/Resource unless you wanna see how fun is it when the game got crash.
